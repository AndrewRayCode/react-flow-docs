The [`<NodeResizer />`](/docs/api/nodes/node-resizer) component can be used to add a resize UI for a custom node. If you want to use it, **you need to install the `@reactflow/node-resizer` package**. This is not part of the `reactflow` package! The `node-resizer` package also exports a `<NodeResizeControl />` component for implementing a custom resizing UI as shown in this example.
